HTTP-запросы

Общение фронта и бэка чаще всего выстроено через HTTP-запросы. При открытии страницы в определённом приоритете начинают отправляться запросы на подгрузку контента.

⚡ Приоритизация HTTP-запросов

Схема приоритизации загрузки ресурсов: HTML → скрипты → стили → шрифты → картинки

📋 Что важно знать в HTTP-запросах

Request / Response
Структура HTTP request (запроса) и HTTP response (ответа) — основа взаимодействия клиента и сервера.
Methods
HTTP methods (методы/глаголы): GET, POST, PUT, PATCH, DELETE и другие — определяют тип действия над ресурсом.
Headers / Body
HTTP headers (заголовки) — метаданные запроса/ответа. HTTP body (тело) — содержимое, передаётся в запросах и ответах.
Status Codes
HTTP status codes (статус-коды) — у ответов. Группы: 1xx, 2xx, 3xx, 4xx, 5xx.
