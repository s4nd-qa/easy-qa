Тестирование фронтенда

Заходим в браузер, открываем сайт — и тестируем. Разбираемся из чего состоит сайт.

🧱 Из чего состоит сайт

Три слоя любого сайта: HTML — каркас, CSS — внешний вид, JS — поведение

HTML
HyperText Markup Language — язык разметки. Структура и содержание: текст, изображения, ссылки.
CSS
Cascading Style Sheets — каскадные таблицы стилей. Внешний вид: расположение, шрифты, цвета, анимации.
JS
JavaScript — язык программирования. Скрипты для фронта: интерактивность, валидация, динамика.
