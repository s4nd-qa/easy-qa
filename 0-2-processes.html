👥 Состав типичной IT-команды

PM
Project Manager — руководит командой и проектом.
Team Lead
Тимлид — руководит конкретной командой, планирует спринты.
BA
Business Analyst — документация, связующее звено между заказчиком и командой.
Designer
UX/UI Designer — рисует макеты, создаёт UI kit для разработчиков.
Frontend
Frontend Developer — разрабатывает UI-интерфейсы, наполняет скриптами.
Backend
Backend Developer — разрабатывает логику систем, пишет API.
QA
Тестировщик — обеспечивает качество. Подключается начиная с требований и заканчивая поддержкой на проде.
DevOps
DevOps Engineer — разворачивает задачи по стендам, поддерживает CI/CD.
