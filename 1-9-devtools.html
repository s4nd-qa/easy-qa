DevTools

DevTools — основной инструмент при тестировании фронтенда. Открой прямо сейчас!

⌨️
Как открыть DevTools
  • Windows: F12 или Ctrl + Shift + I
  • Mac: Fn + F12 или Command + Shift + I

🛠️ Что такое Chrome DevTools

Chrome DevTools — набор инструментов, встроенных в браузер Google Chrome. Позволяют просматривать исходный код сайта, отлаживать HTML, CSS и JavaScript, проверять сетевой трафик, быстродействие и многое другое.

Elements
HTML-структура страницы и CSS-стили. Здесь можно найти конкретный элемент и посмотреть его свойства, отступы, цвета.
Console
Консоль JS. Здесь видны ошибки JavaScript и можно выполнять команды.
Network
Все HTTP-запросы страницы: методы, заголовки, тела запросов/ответов, статус-коды, время загрузки.
Application
Cookie, Local Storage, Session Storage — всё хранилище браузера для текущего сайта.

📊 JSON в DevTools — parsed vs source

В задании 3 тренажёра начнёшь работать с JSON в DevTools. Обрати внимание:

Parsed (отформатированный) — ключи без кавычек, удобно читать

Source (исходный) — ключи в кавычках, это настоящий JSON-формат

⚠️
Важно

JSON в parsed-формате отображается без кавычек у ключей — это только для удобства чтения. Нажми view source чтобы увидеть реальный JSON с кавычками.
