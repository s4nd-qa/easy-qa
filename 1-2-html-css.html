Урок 1-2

HTML и CSS

Для качественного тестирования фронтенда необходимо хорошо разбираться в компонентах HTML и CSS. Умение делать точные ссылки на конкретные элементы — признак профессионала.

⚠️
Зачем тестеру HTML и CSS?

Если нашёл баг с неправильным цветом фона — некорректно написать просто «некорректный цвет фона в блоке». Какого фона? Какого блока? Важно уметь делать точные ссылки на конкретные элементы через DevTools.

ℹ️
Вспомни из вводного модуля

HTML — структура и содержание сайта. CSS — стилизация: расположение, шрифты, размеры, отступы, анимации. Тестерам не нужно уметь верстать — смотри видео без глубокого погружения.

Видео: погружение в HTML и CSS для тестировщика
