UI-элементы, UI-kit

Для большинства привычных UI-элементов придуманы стандартизированные названия. Тестеру необходимо свободно ориентироваться в них — чтобы верно описывать баги и понимать документацию.

🧩 Основные UI-элементы

Окна
Modal (модалка, popup) — всплывающее окно поверх страницы · Alert (алерт) — уведомление · Tooltip (тултип) — подсказка при наведении · Form (форма) — набор полей для ввода данных
Кнопки
Button (кнопка) · Radiobutton (радиокнопка) — выбор одного из нескольких · Checkbox (чекбокс) — галочка · Toggle (тогл) — переключатель вкл/выкл · Slider (слайдер/ползунок)
Поля
Field/Input field (поле ввода) · Search field (поле поиска) · Dropdown (дропдаун) — выпадающий список
Прочее
Progress bar (прогресс-бар) · Loader (лоудер) — индикатор загрузки
Структура
Header (шапка) · Menu (меню) · Sidebar (сайдбар) · Breadcrumbs (хлебные крошки) · Pagination (пагинация) · Footer (футер/подвал)

🎨 UI-kit

На этапе разработки дизайна дизайнеры создают образцы для всех используемых UI-элементов в интерфейсе. Это называется UI-kit — набор UI-элементов.
