🔌 Что такое API

📖
Главное про API

API — это контракт. Договор о том, как «общаться» с конкретным модулем. Одно взаимодействие = API-ручка.

Видео: API простыми словами

⚡ Знакомство со Swagger

Swagger — инструмент документирования и тестирования API.

Swagger Petstore — видны 4 API-ручки: GET, POST, DELETE и загрузка фото

Видео: тестирование API в Swagger

🖱️ Как работать с API-ручкой — пошагово

Разберём на примере ручки POST /pet — добавить нового питомца.

Шаг 1. Нажми Try it out:

Кнопка Try it out активирует режим редактирования запроса

Шаг 2–3. Замени значения и нажми Execute:

Меняем id: 0 → 555, name: "doggie" → "Test", нажимаем Execute

Шаг 4. Проверяем Server Response:

Код 200 и значения id: 555, name: "Test" в ответе — всё прошло успешно

💡
Коды ответов HTTP

200 — успешно · 404 — не найдено · 500 — ошибка сервера
